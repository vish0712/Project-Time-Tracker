Welcome to Project Time Tracker!! Our project time tracker is a productivity tool that helps you keep track of the duration or time taken to cmoplete your projects, be it personal or work!



Features:


  • Add tasks
  • Assign title and relavant project name
  • Save Tasks as they reach completion
  • Pause/Cancel tasks
  • Delete existing tasks
  • Track task currently in progress


Technology used:


  • Javascript
  • HTML
  • CSS


How to use it?


  • Click on track on the top right corner, this will take you to the project time tracker console
  • To add a task click on the "Add Task" button
  • Write the name of the project and click on "Add Project
  • Write the task you wish to accomplish
  • Before clicking on "Add task" select the project you have added from the "Project" drop down menu
  • Click on "Add Task" and your project will be successfully added!
  • To keep track of the duration of the project click on "Change Status" and "Start" from the drop down menu to start recording
  • Once you are done with the project you can stop recording by clicking on "Change status" and then "Mark as completed" from the drop down menu, or if you wish to take a break from the project click on "Stop"
  • If you wish to clear your project console, select the project you want to delete and click on the "Delete Project" button
